# 🚀 JavaScript Assignment Practice 10

A comprehensive collection of **37 interactive JavaScript and DOM assignments** built using **HTML5, CSS3, and Vanilla JavaScript**.

This project focuses on mastering **DOM Manipulation**, **Event Handling**, **Browser APIs**, **LocalStorage**, and building interactive user interfaces through practical assignments.

Developed as part of my **JavaScript learning journey through Sheryians Coding School – Cohort 3.0**.
